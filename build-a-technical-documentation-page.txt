    <section class="main-section" id="Introduction">
      <header>Introduction</header>
      <p>JavaScript is a versatile programming language primarily used for web development.</p>
      <p>It runs in the browser and allows developers to create interactive web pages.</p>
      <code>console.log("Hello, world!");</code>
      <ul>
        <li>Client-side language</li>
        <li>Dynamically typed</li>
        <li>First-class functions</li>
      </ul>
    </section>

    <section class="main-section" id="Variables">
      <header>Variables</header>
      <p>Variables in JavaScript are used to store data values.</p>
      <p>You can declare them using var, let, or const.</p>
      <code>let name = "Nikola";</code>
      <code>const pi = 3.14;</code>
      <ul>
        <li>Use <code>let</code> for reassignable variables</li>
        <li>Use <code>const</code> for constants</li>
      </ul>
    </section>

    <section class="main-section" id="Functions">
      <header>Functions</header>
      <p>Functions are blocks of code designed to perform a particular task.</p>
      <p>You can define them in multiple ways.</p>
      <code>function greet(name) { return "Hello " + name; }</code>
      <ul>
        <li>Can be declared or assigned to variables</li>
        <li>Can be anonymous or named</li>
      </ul>
    </section>

    <section class="main-section" id="Loops">
      <header>Loops</header>
      <p>Loops are used to repeat a block of code multiple times.</p>
      <p>Common types include for, while, and do-while loops.</p>
      <code>for (let i = 0; i < 5; i++) { console.log(i); }</code>
      <ul>
        <li>for loop</li>
        <li>while loop</li>
        <li>do-while loop</li>
      </ul>
    </section>

    <section class="main-section" id="Objects">
      <header>Objects</header>
      <p>Objects are collections of key-value pairs.</p>
      <p>They are fundamental to JavaScript's structure.</p>
      <code>const user = { name: "Nikola", age: 30 };</code>
      <ul>
        <li>Can store any data type</li>
        <li>Used for organizing complex data</li>
      </ul>
    </section>
